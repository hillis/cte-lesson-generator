## Important Rules

### Presentations
- **DO NOT** place the teacher name on any slides
- **DO NOT** place the class name on any slides
- Keep presentations clean and focused on content

### CTE Lesson Plans
- **NO RED TEXT** - All text must be black (red placeholders are automatically converted)
- Files organized into week folders (Week01, Week02, etc.)

### Teacher Handouts
- Professional formatting with navy blue headers and Cambria/Calibri fonts
- Color-coded section headers with icons
- Tables with alternating row colors and styled headers
- Day banners with white text on navy background
- Light blue overview boxes, light yellow teacher note boxes

### Equipment
- Students must pass safety test with **100%** before using equipment
- Required student purchase: SD Card (64GB minimum, UHS-II SDXC V60, V90 recommended)
- Equipment sign-out and sign-in procedures must be followed

## Lesson Plan Format Style

Based on existing lesson plans, use this structure:

### Daily Lesson Plan Structure (90 minutes)
```
## Day X (Day Name) - Lesson Title

### Learning Objectives
- Specific, measurable objectives
- Use action verbs (identify, create, demonstrate, apply)

### Materials Needed
- Equipment and supplies
- Handouts and worksheets
- Technology requirements

### Schedule (90 minutes)

#### Bell Ringer / Warm-Up (5-10 min)
- Quick prompt or review activity
- Gets students thinking about the topic

#### Direct Instruction / Mini-Lecture (15-25 min)
- Key content with bullet points
- Visual examples when possible
- Break into digestible chunks

#### Guided Practice (15-20 min)
- Partner or small group activities
- Teacher circulates and provides feedback
- Structured practice with support

#### Hands-On Activity (25-35 min)
- Independent or collaborative work
- Clear instructions and expectations
- Practical application of concepts

#### Wrap-Up & Reflection (10-15 min)
- Equipment return and check-in
- Reflection prompts in notebooks
- Preview of next lesson
- Exit ticket if needed

### Assessment
- Formative: participation, observation, exit tickets
- Summative: projects, tests, rubrics

### Homework / Extension
- Optional additional practice
- Video links on Schoology
- Preparation for next class
```

## Student Handout Format Style

Student handouts should include:

### Required Elements
1. **Title** - Clear, descriptive heading
2. **Course/Date info** - "Media Foundations | Date | Unit"
3. **Learning Objectives** - "By the end of today's class, you will be able to..."
4. **Step-by-Step Instructions** - Numbered, clear directions
5. **Tables** for comparing options or requirements
6. **Submission Requirements** - File format, naming convention, where to submit

### Formatting Guidelines
- Use tables for structured information
- Include visual organization (headers, bullet points)
- Provide "Required" vs "Recommended" sections where applicable
- **NO Name/Date fields** - All assignments submitted via Schoology
- End with clear deliverables and deadlines

### Student Handout Styling
- Navy banner header with white text title
- Cambria/Calibri fonts matching teacher handouts
- Light blue instruction boxes
- Numbered steps in styled tables with blue number badges
- Vocabulary in alternating-row tables
- Questions with gray answer lines
- Light yellow tips/notes boxes
- Section headers with navy underline borders

### Bell Ringer Slides
- PowerPoint format (.pptx) for Canva upload
- Navy blue background with light blue accent bar
- White "BELL RINGER" title (Cambria, 54pt)
- Week/Day info in light blue
- White rounded content box with navy prompt text
- 16:9 aspect ratio
- Auto-generated from schedule activities named "Bell Ringer" or "Warm Up"
