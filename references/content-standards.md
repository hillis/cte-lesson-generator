# Media Production Foundations - Content Standards

This file contains both Alabama State CTE Standards and Adobe Certified Professional objectives for use in CTE lesson plans.

---

## ALABAMA STATE STANDARDS

**2023 Alabama Course of Study: Arts, A-V Technology, and Communications**

Each standard completes the stem "Students will..."

### Foundational Standards

| Code | Standard |
|------|----------|
| **F1** | Incorporate safety procedures in handling, operating, and maintaining tools and machinery; handling materials; utilizing personal protective equipment; maintaining a safe work area; and handling hazardous materials and forces. |
| **F2** | Demonstrate effective workplace and employability skills, including communication, awareness of diversity, positive work ethic, problem-solving, time management, and teamwork. |
| **F3** | Explore the range of careers available in the field and investigate their educational requirements and demonstrate job-seeking skills including resume-writing and interviewing. |
| **F4** | Advocate and practice safe, legal, responsible, and ethical use of information and technology tools specific to the industry pathway. |
| **F5** | Participate in a Career and Technical Student Organization (CTSO) to increase knowledge and skills and to enhance leadership and teamwork. |

### Content Standards

#### History and Culture

| Code | Standard |
|------|----------|
| **1** | Describe the development of communications technologies from the 19th through 21st centuries, including telegraph, wireless, early radio and television, electronic and satellite news gathering, Internet broadcasting, and emerging technologies. |
| **2** | Outline the evolution of basic modern formats of media consumption and explain their uses. *Examples: film, broadcast television, social media, podcasts, streaming services* |
| **3** | Develop a SWOT (strengths, weaknesses, opportunities, and threats) analysis for future career pathways in the arts and audio/video fields. |

#### Audience and Purpose

| Code | Standard |
|------|----------|
| **4** | Identify the target audiences of various examples of media and the characteristics that make each media type a good match for its audience. |
| **5** | Analyze the tone and purpose of various examples of media. |

#### Production

| Code | Standard |
|------|----------|
| **6** | Summarize the properties of a digital display and outline the advantages and disadvantages of various resolutions. *Examples: pixels, 480p, 720p, 1080p, 4k* |
| **7** | Differentiate among various digital media file types and indicate why each is used. *Examples: JPEG, PNG, TIFF, MP3, WAV, MP4, MOV, AVI* |
| **8** | Identify media production equipment and explain how it is used. *Examples: camera, microphone, lights, supports, software, cables, speakers* |
| **9** | View and critique elements of multimedia performances created for various audiences. *Examples: clarity, diction, eye contact, posture, appearance, audio or video quality* |
| **10** | Create an original media work for a specific purpose and audience, utilizing a variety of media production equipment. *Examples: speech, podcast, commercial, advertisement, musical, broadcast, theatre* |
| **11** | Design technical theatre elements for various media settings, utilizing concepts of visual design. *Examples: stage design, lighting, sound, costumes, props; broadcast, film, theatre, and concert settings* |

#### Audio

| Code | Standard |
|------|----------|
| **12** | Select and configure sound equipment for a small audio project. *Examples: podcast, social media post, streaming, guest speaker, karaoke* |
| **13** | Demonstrate and explain the use of mixing techniques to improve audio quality. *Examples: volume, gain, fading* |

#### Film Production

| Code | Standard |
|------|----------|
| **14** | Identify and describe the basic camera angles used in media production. *Examples: establishing, wide, full, medium, medium close up, close-up, extreme close-up* |
| **15** | Create storyboards to visualize narratives for various multimedia productions. *Examples: short film, commercial, broadcast, music video* |

#### Ethics

| Code | Standard |
|------|----------|
| **16** | Identify and analyze the impact of artificial intelligence and emerging technologies on the media and communications landscape, including both positive and negative effects. |
| **17** | Differentiate between copyrighted and public domain materials and locate examples of each type. |

## ADOBE CERTIFIED PROFESSIONAL OBJECTIVES

**Adobe Certified Professional in Digital Video Using Adobe Premiere Pro (2025 v25.x)**

### 1. Working in the Video Industry

| Code | Objective |
|------|-----------|
| **1.1** | Identify terminology and interpret communication relating to purpose, audience, and project goals for preparing video projects. |
| **1.1.a** | Determine whether content is relevant to the purpose, audience, and audience needs. |
| **1.1.b** | Identify requirements based on how the project will be delivered, distributed, and/or consumed. |
| **1.1.c** | Demonstrate techniques for communicating ideas about project plans with peers and clients. *(shot list, scriptwriting, storyboarding, edit list, transcripts)* |
| **1.1.d** | Demonstrate basic project management and workflow concepts. |
| **1.2** | Demonstrate knowledge of accessibility standards and best practices for digital video content. |
| **1.2.a** | Identify and implement captioning, subtitles, audio descriptions, and other assistive elements. |
| **1.2.b** | Apply techniques for clear and legible text overlays, proper color contrast, and audio clarity. |
| **1.3** | Assess intellectual property rights, permissions, and licensing requirements. |
| **1.3.a** | Determine legal and ethical considerations for using third-party content. *(copyright, fair use, AI-generated content)* |
| **1.3.b** | Determine when and how to obtain permission to use images, audio, or video. *(talent release, licensing, attribution)* |
| **1.4** | Demonstrate an understanding of video and audio terminology and knowledge of editorial practices and principles of film form. |
| **1.4.a** | Define video and audio terminology. *(frame rate, aspect ratio, codecs, aperture, shutter speed, ISO, clipping, levels, foley)* |
| **1.4.b** | Use visual storytelling techniques and editorial principles and guidelines. *(L and J cuts, match cuts, montage, jump cut, transitions)* |
| **1.4.c** | Define common film form terms and principles. *(rule of thirds, depth of field, shot types: CU, ECU, MCU, MS, WS, LS, OTS, POV)* |

### 2. Project Setup and Interface

| Code | Objective |
|------|-----------|
| **2.1** | Create projects and sequences. |
| **2.1.a** | Choose appropriate project settings. *(file location, sequence settings, scratch disks)* |
| **2.1.b** | Create a sequence that matches the delivery requirements. *(aspect ratio, frame size, sample rate)* |
| **2.2** | Navigate, organize, and customize the application workspace. |
| **2.2.a** | Identify, navigate, and manipulate elements of the Premiere Pro interface. *(panels, timeline, monitors)* |
| **2.2.b** | Organize and customize the workspace. |
| **2.2.c** | Configure application preferences. *(Auto Save, Media Cache, Timeline settings)* |
| **2.3** | Use non-visible design tools in the interface to aid in the video editing workflow. |
| **2.3.a** | Controlling and working with the timeline and media. *(JKL navigation, keyboard shortcuts)* |
| **2.3.b** | Use markers. |
| **2.3.c** | Use guides, rulers, and safe margins. |
| **2.4** | Import assets into a project. |
| **2.4.a** | Import media from various sources. *(importing files, Media Browser, proxies)* |

### 3. Organizing Video Projects

| Code | Objective |
|------|-----------|
| **3.1** | Manage assets in a Premiere Pro project. |
| **3.1.a** | Organize assets in the Project panel. *(bins, metadata, relinking, Project Manager)* |
| **3.2** | Manage clips, tracks, and sequences. |
| **3.2.a** | Configure sequences and clips in a project. *(nesting, syncing audio, Multi-Camera)* |
| **3.2.b** | Configure tracks for playback and display. *(locking, targeting, mute, solo)* |

### 4. Creating and Modifying Elements

| Code | Objective |
|------|-----------|
| **4.1** | Assemble a video sequence using a variety of methods and panels. |
| **4.1.a** | Create a rough cut. *(In/Out points, insert, overwrite, lift, extract)* |
| **4.2** | Perform text-based editing. |
| **4.2.a** | Create a transcript from a clip. |
| **4.2.b** | Modify videos using Text-Based Editing. |
| **4.2.c** | Export transcripts. |
| **4.3** | Add text and graphics and modify their properties. |
| **4.3.a** | Add text and shapes to a sequence. *(Type tools, Graphics Templates, MOGRTs)* |
| **4.3.b** | Use controls to modify text and graphic appearance. *(font, alignment, fill, stroke)* |
| **4.3.c** | Add and modify captions. |
| **4.4** | Transform footage within a project. |
| **4.4.a** | Resize clips. *(scale, rotation, aspect ratio, Generative Expand)* |
| **4.4.b** | Trim and refine clips. *(Rolling Edit, Ripple Edit, Rate Stretch, Scene Edit Detection)* |
| **4.4.c** | Modify video clip settings. *(speed, time remapping, freeze frame)* |
| **4.5** | Use effects to modify clips in a sequence. |
| **4.5.a** | Use color correction methods and tools. *(Lumetri Color, LUTs)* |
| **4.5.b** | Apply effect presets. |
| **4.5.c** | Apply effect presets on multiple clips. *(adjustment layers, nesting)* |
| **4.5.d** | Composite video clips. *(keying, opacity, masking, alpha channel)* |
| **4.6** | Use keyframes to control video properties over time. |
| **4.6.a** | Modify attributes in the Effect Controls Panel and Properties Panel. |
| **4.6.b** | Apply and adjust video effects in the Timeline. *(motion, opacity)* |
| **4.7** | Manage audio in a sequence. |
| **4.7.a** | Apply effects presets. *(transitions, FX badges)* |
| **4.7.b** | Add audio to a sequence. *(synchronizing, linking, mono/stereo)* |
| **4.7.c** | Adjust audio on the timeline or using the Essential Sound panel. *(Enhance Speech, Auto-Match, Ducking)* |
| **4.7.d** | Adjust audio. *(gain, conforming, fade handles)* |
| **4.7.e** | Record audio to the timeline. *(Voice-over record)* |

### 5. Publishing Digital Media

| Code | Objective |
|------|-----------|
| **5.1** | Prepare a sequence for export. |
| **5.1.a** | Check a sequence for errors and project specifications. *(gaps, audio levels, safe margins)* |
| **5.2** | Prepare sequences for distribution to multiple platforms. |
| **5.2.a** | Create a new version of an existing sequence using Auto Reframe. |
| **5.3** | Export and archive video and audio sequences. |
| **5.3.a** | Export a single frame, a clip, a range of a sequence, or an entire sequence. *(codecs, Quick Export)* |
| **5.3.b** | Export using Adobe Media Encoder. *(batch processing, multiple formats)* |
| **5.3.c** | Archive a project. *(Project Manager, collect files, consolidate)* |

## CTE LESSON PLAN FORMAT

When filling the **Content Standard(s)** field in CTE lesson plans, use this format:

```
ALABAMA STATE STANDARDS:
Standard 14: Identify and describe the basic camera angles used in media production.
Standard 8: Identify media production equipment and explain how it is used.
F1: Incorporate safety procedures in handling, operating, and maintaining tools and machinery.

ADOBE CERTIFIED PROFESSIONAL:
1.4.c: Define common film form terms and principles (shot types, rule of thirds, depth of field).
1.4.a: Define video and audio terminology (frame rate, aspect ratio, aperture, shutter speed, ISO).
```
